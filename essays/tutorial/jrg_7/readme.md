题目1： HTML5是什么？有哪些新特性？有哪些新增标签？如何让低版本的 IE 支持 HTML5新标签

HTML5是应用超文本标记语言（HTML）第五次重大修改产生的规范。

特性:
Semantic :
语义化网页意义和结构

本地存储特性:  
APP cache 、IndexDB

MULTIMEDIA:
audio video等

三维、图形及特效特性:
svg canvas webGL

浏览器层面:

webRTC

Server-Sent Event和WebSockets


新增标签：header nav footer article section aside dialog audio vedio
